# CodeAlpha E-commerce Store

A full-stack e-commerce web application built using React, Node.js, Express, and MongoDB.

## Features
- Product listing
- Search products
- Add to cart
- Cart management
- Checkout system
- Order success animation
- Modern dark UI

## Tech Stack
Frontend:
- React
- Tailwind CSS
- Framer Motion

Backend:
- Node.js
- Express.js
- MongoDB

## Project Setup

### Backend
cd server  
npm install  
npm start  

### Frontend
cd client  
npm install  
npm run dev  

## Demo Flow
Home → Products → Cart → Checkout → Order Success
